# 🚀 LaymanInvestor V3.0 - Quick Start Guide

## ⚡ 3-Minute Setup

### Step 1: Database Migration (2 minutes)
1. Open https://supabase.com
2. Go to your project → SQL Editor
3. Copy all from `DATABASE_MIGRATION_V3.sql`
4. Paste and click "Run"
5. Wait for "Success" message

### Step 2: Verify Backend (1 minute)
Open these URLs in browser:

```
https://laymaninvestor-backend.onrender.com/health
https://laymaninvestor-backend.onrender.com/api/news/market
https://laymaninvestor-backend.onrender.com/api/stocks/ipos
```

All should return JSON data.
